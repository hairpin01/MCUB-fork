# Telegram UserBot

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте бота через @BotFather и получите BOT_TOKEN
2. Получите API_ID и API_HASH на https://my.telegram.org
3. Откройте `userbot.py` и замените:
   - `YOUR_API_ID` - ваш API ID
   - `YOUR_API_HASH` - ваш API Hash
   - `YOUR_BOT_TOKEN` - токен бота от @BotFather

## Запуск

```bash
python userbot.py
```
