# Каталог модулей для Mitrich UserBot

Здесь размещаются официальные модули для юзербота.

## Установка модуля

Используйте команду в Telegram:
```
.dlm название_модуля
```

Например:
```
.dlm example
```

## Доступные модули

### example.py
Простой пример модуля с командами:
- `.dice` - бросок кубика
- `.flip` - подбросить монетку

## Создание своих модулей

1. Создайте файл `ваш_модуль.py`
2. Добавьте функцию `register(client)`
3. Зарегистрируйте обработчики команд
4. Загрузите в эту папку на GitHub

### Пример структуры:

```python
from telethon import events

def register(client):
    @client.on(events.NewMessage(outgoing=True, pattern=r'^\.команда$'))
    async def handler(event):
        await event.edit('Ответ')
```

## Загрузка в каталог

1. Форкните репозиторий
2. Добавьте свой модуль в `modules_catalog/`
3. Создайте Pull Request
4. После проверки модуль будет доступен всем
